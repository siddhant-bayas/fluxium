# Community

## Contributing

See [CONTRIBUTING.md](../../CONTRIBUTING.md) at the project root for development setup, linting, testing, and PR guidelines.

## Security

To report a security vulnerability, please open a GitHub issue with the `security` label or contact the maintainer directly. Do not disclose security issues publicly until a fix is released.

## Code of Conduct

Be respectful, constructive, and inclusive. This is a small project — kindness goes a long way.
